# Level 8: One-Room Platformer

## Fantasy
Navigate a single room, jumping over spikes to reach the goal.

## Expected Mechanical Lesson
**Sub-tile Physics and AABB Collision.** 
Transitioning from discrete tile grids and isolated state machines into vector-based spatial physics. The lesson focuses on maintaining a deterministic `tick(deltaSeconds)` simulation that strictly handles velocity, gravity, and Axis-Aligned Bounding Box (AABB) collision resolution independently from the renderer.
