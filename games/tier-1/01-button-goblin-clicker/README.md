# Button Goblin Clicker

## Purpose
A minimal clicker game to demonstrate state management, simple interactivity, and the test-first integration of a pure simulation module with a Phaser/DOM frontend.

## How to run v0.1

If you haven't already, install dependencies from the repository root:
```bash
pnpm install
```

Run the development server for this game:
```bash
pnpm --filter tga-01-button-goblin-clicker dev
```

Run tests:
```bash
pnpm --filter tga-01-button-goblin-clicker test
```

Build the game:
```bash
pnpm --filter tga-01-button-goblin-clicker build
```
