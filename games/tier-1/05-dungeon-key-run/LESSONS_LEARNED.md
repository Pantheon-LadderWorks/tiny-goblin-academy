# Level 5 Lessons Learned

1. **Hazard Relevance**: A hazard is not valid just because it exists. It must intersect the critical path or meaningfully shape route planning. The initial Goblin was a decorative hazard until the map was rebuilt into a chokepoint.
2. **Ledger Numbering**: If a ledger displays newest-first, the visible numbering must still preserve historical step order.
