# Code of Conduct

We want Tiny Goblin Academy to be a positive, experimental learning space. Please adhere to the following simple guidelines when participating in issues, discussions, or pull requests:

* **Be respectful:** Treat everyone with kindness and professional courtesy.
* **No harassment or hate speech:** We do not tolerate harassment, discrimination, or abusive language of any kind.
* **No spam:** Keep discussions focused on the project.
* **Keep feedback constructive:** If you have critiques of the architecture or the AI methodology, phrase them constructively.

This is Kryssie’s project space. Maintainers hold the right to remove or block anyone exhibiting disruptive behavior.
