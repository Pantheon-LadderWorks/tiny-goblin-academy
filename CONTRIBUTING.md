# Contributing to Tiny Goblin Academy

Thank you for your interest in Tiny Goblin Academy! 

Before contributing, please note that this is an **experimental learning and project archive**, not a normal open contribution free-for-all yet. It is the direct output of the AI Game Studio Ladder methodology.

### What We Welcome
* Issues and bug reports regarding the Tier 1 games or the upcoming Hub.
* Documentation corrections (typos, clarity improvements).
* Small, scoped suggestions for the existing Tier 1 architecture.

### What Requires Prior Discussion
Please open an issue to discuss the following *before* submitting a Pull Request:
* Major feature proposals or architectural rewrites.
* New game submissions.
* Monetized forks, mobile ports, or rebrands.

### Development Guidelines
* **Use pnpm:** We use `pnpm` exclusively for package management, not `npm` or `yarn`.
* **No Bloat:** Do not commit `node_modules`, `dist`, `.vite`, generated build outputs, secrets, or private seed concepts.
* **Level 1 Restoration:** Do not attempt to rebuild Level 1 (`01-button-goblin-clicker`). It is intentionally deferred until the hub/runtime package architecture supports it.
* **Respect the Process:** Respect the project scope, the AI/Human collaboration history, and the explicit boundaries set by the maintainers.
