# Tiny Goblin Academy Assets

This folder contains the visual assets, sprite sheets, and UI elements used by the Tiny Goblin Academy games and hub.

## Important Notes

* **Concept vs Production:** Many assets here are marked as `-concept-v0.1`. These are initial iterations that may contain text labels or mixed perspectives. They will be superseded by clean, production-ready `-v0.1` files over time.
* **No Manual Cutting Doctrine:** Do not manually crop or slice these sheets in an image editor. We rely on engine configuration via `source rects` to draw individual frames.
* **Manifests Incoming:** In the future, `.manifest.json` sidecar files will be added to explicitly declare grid sizing, frame counts, anchors, and animation sequences.
