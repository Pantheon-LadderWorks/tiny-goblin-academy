# Tiny Goblin Academy — Tier 2 Seed Plan

## Purpose
Capture early Tier 2 thinking without authorizing implementation.

## Working Idea
Tier 2 may include:
* five selected Tier 1 games upgraded to bounded v0.2;
* five new games introducing new architectural lessons.

## Why This Is Different From Tier 1
Tier 1 taught one-loop first playable discipline.
Tier 2 should teach:
* evolving existing code safely;
* refactoring without breaking the loop;
* asset/animation integration;
* stronger AI/pathing/procedural/state lessons if carefully bounded;
* packaging and product lifecycle awareness.

## Candidate v0.2 Upgrade Pool
*(Possible candidates, not decisions)*
* **Level 4 Card Goblin Duel:** Add a second card type or a basic deck shuffle.
* **Level 5 Dungeon Key Run:** Add multiple keys/doors or simple enemy patrol pathing.
* **Level 7 Pet Campfire:** Add a second survival meter or an interactive tool.
* **Level 8 One-Room Platformer:** Add a double jump or a moving platform.
* **Level 9 Top-Down Slime Quest:** Add simple inventory or multiple enemy types.
* **Level 10 Mini Settlement Sim:** Add a second structure type or a tech upgrade.

## Candidate New Game Slots
*(Possible categories for new architectural lessons)*
* pathfinding microgame;
* procedural room generator;
* dialogue choice microgame;
* inventory/equipment microgame;
* rhythm/timing microgame;
* stealth/line-of-sight microgame;
* physics puzzle microgame;
* save/load/profile microgame.

*(Note: All specific game concepts remain undecided.)*

## Guardrails
* No Tier 2 implementation yet.
* No automatic expansion of all Tier 1 games.
* v0.2 requires a fresh contract.
* Level 11 requires a fresh contract.
* Hub and visual pass may happen before Tier 2.
