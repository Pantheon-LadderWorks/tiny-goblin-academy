# Tiny Goblin Academy — H2.5b Transparent Sheet Cleanup Report

## Context & Motivation
During Phase H2.5, a region-edge cleanup script was used to convert baked checkerboard into transparency. While safe and conservative (it only acted on pixels connected to the edges of the icon regions), it left artifacts and "hard crumbs" where checkerboard pixels existed deeper within the irregular source sheet layout. 

Phase H2.5b successfully demonstrated that a **full-sheet border-connected flood fill** is the superior method for converting fake-transparent PNG sheets. By seeding from the very edges of the entire image and looking for low-saturation gray-like pixels, the algorithm successfully converted the fake checkerboard background into a true PNG alpha channel without harming the vibrant colors or intentional internal gray pixels of the game icons.
