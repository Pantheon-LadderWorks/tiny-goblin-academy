# Tiny Goblin Academy Asset Sheet Intake Checklist

## Sections

* [ ] Sheet Identity
* [ ] Intended Use
* [ ] Source Status
* [ ] Inspection
* [ ] Alpha / Fake Transparency Audit
* [ ] Layout Classification
* [ ] Static or Animation Classification
* [ ] Region / Frame Mapping
* [ ] Derived Cleanup Decision
* [ ] Validator Requirement
* [ ] Wiring Readiness
* [ ] Screenshot / Contact Sheet Evidence
* [ ] Human Review Gate
* [ ] Promotion Status

## Promotion Statuses

* `concept`
* `registered`
* `inspected`
* `mapped`
* `derived`
* `validated`
* `wired`
* `human-review-passed`
* `needs-revision`

## Important Notes

* Do not wire a sheet before it reaches at least `mapped` and `validated`.
* Do not treat generated sheets as production-ready.
* Do not trust “transparent-looking” backgrounds without an alpha audit.
* Do not process animation sheets in bulk without a pilot test.
